Koble din Sungrow vekselretter til Homey og få detaljert innsikt i din solenergiproduksjon og batteribruk.
Overvåk din nåværende strømproduksjon, samt din daglige, månedlige, årlige og totale produksjon. Med Sungrow-appen for Homey kan du bruke disse dataene til å optimalisere energibruken din, opprette Flows og holde oversikt over systemets ytelse – alt på ett sted.
